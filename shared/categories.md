# GitHub Stars 默认分类

`gh-stars-organizer` 默认管理以下 10 个 GitHub Lists。

| List | 用途 |
|---|---|
| `AI 编程与 Agent IDE` | Claude Code、Codex、OpenClaw、Cursor、AionUi |
| `Agent 框架与多 Agent` | Hermes、Daytona、多 Agent 框架 |
| `MCP 与 Skills` | MCP server、Skills、agent 能力包 |
| `RAG 与文档处理` | RAG、OCR、PDF、Markdown、知识库 |
| `Prompt 与 LLM 工具` | Prompt、system prompts、模型工具 |
| `自动化与爬虫` | 浏览器自动化、爬虫、平台采集 |
| `UI 与应用模板` | Tauri、NiceGUI、设计 Skill、前端模板 |
| `开发基础设施与 CLI` | CLI、终端、打包、开发工具 |
| `内容与媒体 AI` | 图像、视频、语音、内容生产 |
| `待复核 Review Later` | 低 star、重复、fork、过时、用途不明 |

## 管理边界

- 只管理上述 10 个 Lists。
- 用户已有其他 Lists 必须保留。
- 更新仓库 Lists 时，必须保留非本 skill 管理的旧 List 关系。
- 默认不取消 star。
